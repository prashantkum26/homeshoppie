# HomeShoppie Category Images Generation Guide

This directory contains category images for the HomeShoppie e-commerce platform. Each category needs a professional, high-quality image that represents traditional Indian food products.

## Image Specifications

- **Dimensions**: 400x400px (square format)
- **File Format**: PNG or WebP
- **Style**: Professional food photography with warm, inviting aesthetic
- **Background**: Clean, minimal background that complements the category color theme
- **Lighting**: Soft, natural lighting that highlights the products

## Usage in Application

Once images are generated and placed in this directory, update the category pages to display these images instead of emoji icons. The images will be accessible at:

- `/images/category/ghee.png`
- `/images/category/oils.png`
- `/images/category/sweets.png`
- `/images/category/namkeen.png`
- `/images/category/pooja-items.png`

## Image Generation Tools Recommended

1. **DALL-E 3** (OpenAI) - Best for food photography realism
2. **Midjourney** - Excellent for artistic food presentation
3. **Stable Diffusion** - Good free alternative with custom models
4. **Adobe Firefly** - Professional grade for commercial use

## Quality Checklist

- [ ] High resolution (400x400px minimum)
- [ ] Professional lighting and composition
- [ ] Colors match category theme
- [ ] Clear, appetizing presentation
- [ ] Consistent style across all categories
- [ ] Commercial usage rights confirmed
- [ ] Optimized file size for web
- [ ] PNG format with transparency support

## Alternative Sources

If AI generation doesn't meet requirements:

1. **Unsplash** - Free high-quality stock photos
2. **Pexels** - Free stock photography
3. **Shutterstock** - Premium stock photos
4. **Local Photography** - Hire professional food photographer

## Implementation Notes

After generating images, update the category page component to use real images instead of emoji placeholders. The images should enhance the user experience and make the categories more visually appealing and professional.
